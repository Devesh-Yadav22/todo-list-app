# 📝 Smart To-Do List App

> 🚀 **[Live Demo: Click here to view the To-Do List App on Render!](https://todo-list-app-akwp.onrender.com)**

A functional, modern, and persistent To-Do List application designed to help users manage tasks efficiently. This project features task prioritization, deadline tracking, and local data persistence.

<br />

## ✨ Features
- **Task Management:** Create, edit, and delete tasks with ease.
- **Priority Levels:** Categorize tasks by priority (Low, Medium, High) with visual color-coding.
- **Deadline Tracking:** Set due dates for each task to stay on top of your schedule.
- **Data Persistence:** Uses `sessionStorage` to ensure your tasks remain available even after page refreshes.
- **Clean UI:** Responsive and minimalist design for a clutter-free productivity experience.
